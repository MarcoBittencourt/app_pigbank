# 🐷 PigBank

Projeto desenvolvido para a disciplina de **Desenvolvimento de Aplicação Web**, ministrada pelo professor **Victor Brunno**, que também auxiliou no desenvolvimento e orientação do trabalho.

## Sobre o projeto

O PigBank é uma aplicação web de controle financeiro pessoal criada com o objetivo de ajudar usuários a acompanharem suas receitas, despesas e metas financeiras de forma simples e organizada.

A proposta do sistema é oferecer uma maneira prática de registrar movimentações financeiras, categorizar gastos, acompanhar o saldo disponível e visualizar informações importantes por meio de relatórios e gráficos. Dessa forma, o usuário consegue ter uma visão mais clara de sua vida financeira e tomar decisões com mais segurança.

O projeto foi desenvolvido como atividade acadêmica, aplicando conceitos estudados durante a disciplina, como desenvolvimento front-end, integração com banco de dados, autenticação de usuários e organização de aplicações web modernas.

## Funcionalidades

* Cadastro e gerenciamento de receitas e despesas;
* Criação e personalização de categorias;
* Controle de transações recorrentes;
* Definição e acompanhamento de metas financeiras;
* Relatórios e indicadores financeiros;
* Sistema de autenticação de usuários;
* Dashboard com resumo financeiro.

## Tecnologias utilizadas

* React
* TypeScript
* Vite
* Tailwind CSS
* Supabase

## Executando o projeto

```bash
npm install
npm run dev
```

Após iniciar a aplicação, acesse:

```text
http://localhost:5173
```
